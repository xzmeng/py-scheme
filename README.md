# Scheme Interpreter written in Python
This repo is my implementation of the Scheme interpreter of the final project in [CS61A](https://cs61a.org/).
## Usage

- REPL
```bash
python scheme.py
```
- .scm file
```python
python scheme.py a.scm
```
